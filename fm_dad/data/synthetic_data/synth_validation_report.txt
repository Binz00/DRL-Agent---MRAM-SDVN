===== NOTE ON IGH GATE =====
IGH is different from SP/ALS/FS. Its detection gate is NOT one single feature.
It requires TWO features together: high PDRVar AND high CoordScore (report
Definition 3 / Eq. 3.19-3.22). So the simple one-feature gate check above is
misleading for IGH. Verified separately:
  ATTACKER : PDRVar mean=0.132, CoordScore mean=0.998
  INNOCENT : PDRVar mean=0.001, CoordScore mean=0.274
  Attacker rows passing BOTH gate conditions: 2877/3000 (96%).
  (The ~4% that do not pass are early cycles where the window is not yet full.)
The false-positive victim group (Group 2) has LOW PDRVar (stable dropping) OR
low rho_recv, so the agent must learn that a low FFc mean alone is NOT proof of
IGH — only the ON/OFF variance + coordination signature is.
